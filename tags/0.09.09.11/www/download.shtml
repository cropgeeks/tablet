Download Tablet

The most recent release of Tablet is 0.09.09.11 (11th September 2009).

Please use the links below to download the Tablet installer most suitable for your operating system. Tablet is currently available for:

    Windows (32 bit) or Windows (64 bit)
    Linux (32 bit) or Linux (64 bit)
    Apple Mac OS X

Follow the instructions provided by the graphical installer to install Tablet. Note that Tablet should not require administrator privileges to install or run, but if you have any problems, don't hesitate to contact us at bioinf@scri.ac.uk.

Tablet is developed on Windows XP SP3, and gets additional regular testing on OS X 10.6, Ubuntu Linux, and Centos 5.
