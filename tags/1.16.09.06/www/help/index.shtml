Welcome to Tablet Help

To get help when using Tablet, select the Help button from the far right-hand side of the ribbon bar. Help is also available from any of Tablet's dialogs by clicking the Help button if visible. Note that help is only available while Tablet is running on a computer with an active internet connection.

If you cannot find what you are looking for within this documentation, more help and support is available by emailing us at tablet@hutton.ac.uk.
