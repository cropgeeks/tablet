Open Assembly

The Open Assembly dialog is used to load assemblies into Tablet for viewing.

Access the dialog by selecting either Open Assembly from the Data tab of the Ribbon Bar or by selecting Open from the Application Menu.

Tablet can currently view assemblies which are stored in the ACE, AFG, MAQ, or SOAP file formats. To choose which type of assembly you wish to open, simply select your desired file format and then provide the input file (or files) required.
