Download Tablet

The most recent release of Tablet is 1.13.05.17 (17th May 2013).

Please use the links below to download the Tablet installer most suitable for your operating system. Tablet is currently available for:

    Windows (32 bit) or Windows (64 bit)
    Linux (32 bit) or Linux (64 bit)
    Mac OS X (note requirements below)

A Java Web Start version of Tablet also exists. Instructions for using it can be found in the Tablet FAQ.

Follow the instructions provided by the graphical installer to install Tablet. Note that Tablet should not require administrator privileges to install or run, but if you have any problems, don't hesitate to contact us at tablet@hutton.ac.uk.

Tablet is developed on Windows 7, and gets additional regular testing on Mac OS X and Centos.

Archived Versions

You can download a limited selection of older copies of Tablet from our archive. If you do need to return to an older version (for whatever reason), please let us know why.

Additional downloads

  • coveragestats.py - python script that summarizes the coverage export file from Tablet. Provided by Zach Foster and Aaron Liston, Department of Botany & Plant Pathology, Oregon State University.

OS X Requirements

Tablet relies on Java 7 which is only officially supported on OS X 10.7.3 or later, however you may still be able to run it on earlier versions. We bundle Tablet with an appropriate Java runtime so you shouldn't have any problems starting it, but we have had at least one report of various degrees of graphical corruption when running it on older OS X versions.
