Download Tablet

The most recent release of Tablet is 1.10.05.18 (18th May 2010).

Please use the links below to download the Tablet installer most suitable for your operating system. Tablet is currently available for:

    Windows (32 bit) or Windows (64 bit)
    Linux (32 bit) or Linux (64 bit)
    Apple Mac OS X
    Solaris (Sparc)

A Java Web Start version of Tablet also exists, and can be made available on request in certain circumstances, however this is not something we can officially support.

Follow the instructions provided by the graphical installer to install Tablet. Note that Tablet should not require administrator privileges to install or run, but if you have any problems, don't hesitate to contact us at tablet@scri.ac.uk.

Tablet is developed on Windows 7, and gets additional regular testing on Windows XP SP3, OS X 10.6, Ubuntu Linux, and Centos 5.
